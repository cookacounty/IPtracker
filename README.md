IPtracker
=========

Track em, find em, kill em.


## Setup Instructions


### 1. Install Rails:


[Rails installer](http://railsinstaller.org/en)

### 2. Install Aptana:
[Aptana](http://www.aptana.com/products/studio3/download)
 
### 3. Install Bundle:
    bundle install
    
### 4. Generate Database:
    bundle exec rake db:migrate

### 5. Populate Database:
    bundle exec rake db:populate
    
### 6. Start Server:
    bundle exec rails s
